Rappel Markdown utilises pour organiser et structurer un texte

1. Titres (hierarchie)

Permettent de structurer comme un plan (chapitres, sous-parties).

# Titre niveau 1
## Titre niveau 2
### Titre niveau 3
#### Titre niveau 4

Exemple rendu :

Titre 1

Titre 2

Titre 3

Usage :

  • # - titre principal
  • ## - section
  • ### - sous-section

2. Listes

Liste a puces

- element 1
- element 2
  - sous-element

Resultat :

  • element 1
  • element 2
  • sous-element

Liste numerotee

1. premier
2. deuxieme
3. troisieme

Resultat :

  1. premier
  2. deuxieme
  3. troisieme

3. Mise en forme du texte

*italique* ou _italique_
**gras** ou __gras__
***gras + italique***

Resultat :

  • italique
  • gras
  • gras + italique

4. Code

Code inline

`code`

Exemple : SELECT * FROM table

Bloc de code

```sql
SELECT * FROM stock;
```

Resultat :

SELECT * FROM stock;

5. Liens

[Texte du lien](https://example.com)

Exemple : Google

6. Images

![alt](url_image)

7. Citations

> Ceci est une citation

Resultat :

Ceci est une citation

8. Tableaux

| Produit | Stock |
|--------|------|
| A      | 100  |
| B      | 50   |

Resultat :

Produit Stock
A 100
B 50

9. Ligne de separation

---

Resultat :

10. Checklist (tres utile)

- [ ] A faire
- [x] Fait

Resultat :

  • [ ] A faire
  • [x] Fait

11. Synthese structuree

Element Symbole principal
Titre #
Liste - ou 1.
Gras **
Italique *
Code `
Lien [ ] ( )
Citation >
Tableau |
Separateur ---

12. Version simple (niveau 10 ans)

Markdown = ecrire du texte avec des petits symboles pour le rendre joli.

  • # - faire un titre
  • - - faire une liste
  • ** - mettre en gras
  • ` - montrer du code

C'est comme donner une forme lisible a tes idees sans utiliser Word.
